## Project Context

- **Project:** APEX — Autonomous Stock-Picking Agent
- **User:** Jan G.
- **Created:** 2026-04-13
- **Stack:** React (frontend), Node.js (backend), Azure Web App (hosting)
- **Description:** An autonomous stock-picking agent that continuously analyzes stocks and ETFs across US, Europe, and Asia markets. Performs qualitative/trend-based analysis (P/E ratios, macro trends, Google Trends, social sentiment, financial news). Builds and maintains a virtual portfolio with conviction-based trading. Logs all decisions with rationale. Tracks performance with visualizations. Learns from past decisions to improve over time.
